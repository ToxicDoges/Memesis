# Memesis
Our patroness memesis


## Installation

1. Clone the repository:

bash
git clone https://github.com/ToxicDoges/Memesis.git


2. Change directory to the project folder:

```bash
cd Memesis
```

3. Install dependencies:

```bash
go mod download
```

## Usage

1. Build the project:

```bash
go build
```

2. Run the executable:

```bash
./Memesis
```
